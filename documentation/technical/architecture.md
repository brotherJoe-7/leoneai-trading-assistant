# System Architecture

## Overview
LeoneAI is a distributed system with three main components:
1. AI Engine (24/7 analysis)
2. Backend API (FastAPI)
3. Frontend Clients (Web + Mobile)

## Technology Stack
- **AI Engine**: Python, pandas, ta-lib
- **Backend**: FastAPI, PostgreSQL, Redis
- **Frontend Web**: React.js, Vite, Chart.js
- **Mobile**: React Native, Expo
- **Infrastructure**: Docker, Nginx
