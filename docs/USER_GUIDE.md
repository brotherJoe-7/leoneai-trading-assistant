## Signals & Analysis

Signals show AI-generated trading recommendations for forex and crypto markets relevant to Sierra Leone.

| Column         | Meaning                                          |
| -------------- | ------------------------------------------------ |
| **Pair**       | The asset being traded (e.g. USD/SLL, BTC/USD)   |
| **Action**     | BUY / SELL / HOLD                                |
| **Confidence** | How certain the AI is (higher = stronger signal) |
| **Risk Level** | LOW / MEDIUM / HIGH                              |
| **Target**     | Price to aim for when taking profit              |
| **Stop Loss**  | Price to exit at to limit losses                 |
